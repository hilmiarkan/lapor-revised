# Lapor-Revised
A web-based complaint management system (Sistem Informasi Pelaporan) designed to bridge communication between citizens (masyarakat), agencies (instansi), and administrators.

- Tech Stack: HTML5, Vanilla CSS, Vanilla JavaScript,PHP (Native), MySQL
- 3 Role user, Admin, Instansi, Masyarakat
- CRUD of post, Register & Login feature

### Masyarakat (Citizen)
*   **Authentication:** Register and log in.
*   **Submit Reports:** Create new complaints/reports with image attachments.
*   **Anonymity:** Option to hide identity when submitting a report.
*   **Tracking:** View the real-time status of submitted reports.
*   **Interaction:** Add comments to reports.

### Instansi (Agency)
*   **Dashboard:** View all assigned or relevant reports.
*   **Status Management:** Update report statuses (e.g., *menunggu*, *diverifikasi*, *ditindaklanjuti*, *selesai*).
*   **Communication:** Reply to citizen reports via comments.

### Admin
*   **Global Overview:** Access and manage all reports across the system.
*   **Verification:** Review and verify incoming reports before they are processed by agencies.
*   **System Moderation:** Full control over reports and comments.
